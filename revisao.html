Capítulo 12

Box Model

No CSS, todo elemento é uma caixa retangular composta por quatro camadas.

As quatro camadas

12 · 01
  • Margin — espaço externo, separa elementos vizinhos
  • Border — borda visível ao redor do padding
  • Padding — espaço interno entre borda e conteúdo
  • Content — o conteúdo real (texto, imagem)

Aplicação prática

12 · 02 
.card {
  width:   300px;            // largura do conteúdo
  padding: 20px;             // espaço interno
  border:  2px solid #38bdf8; // borda visível
  margin:  16px;             // espaço externo
}
A largura total do elemento será: 2 × margin (32) + 2 × border (4) + 2 × padding (40) + width (300) = 376px. Use box-sizing: border-box para incluir border e padding na width.

Pseudo-classes

13 · 02

  background: #f1f5f9;
}
  • :hover — ao passar o mouse
  • :focus — elemento em foco
  • :checked — checkbox marcado
  • :first-child — primeiro filho
  • :nth-child(n) — por posição
  • :not(seletor) — negação
