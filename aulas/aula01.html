O que é HTML?

  • Hyper Text Markup Language
  • Elemento básico da construção de páginas web
  • HTML não é uma linguagem de programação

O que é Programação Web?

Programação web é a arte de criar e desenvolver páginas e aplicações para a internet. Ela envolve diversas tecnologias para estruturar, estilizar e dar interatividade aos conteúdos.

Da criação de sites simples a aplicações complexas e dinâmicas!
